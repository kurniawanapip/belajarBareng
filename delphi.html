    4.  Cara menghubungkan delphi dan database (Microsoft acces).
        • Kembali ke form delphi, Klik pada tool adoconnection, lalu pada properties pilih connection string, klik tanda seperti ini   , kemudian akan muncul table seperti di bawah !!
        
        • Lalu klik build – pilih Microsoft office 12.0 acces database engine OLE DB provider – lalu next
        
        • Kemudian akan muncul table
        
    Tulis direktori database nya pada kolom datasource, 
    untuk contoh direktori yang saya simpan yaitu (D:\AMIK\pemrograman delphi\data perpustakaan) dan tinggal di tambah di belakangnya \perpustakaan.accdb

    Sehingga menjadi seperti ini D:\AMIK\pemrograman delphi\data perpustakaan\perpustakaan.accdb

    Dan kalau untuk yang anda buat menjadi seperti ini D:\ data perpustakaan\perpustakaan.accdb

    kemudian pilih test connection,
    Jika muncul test connection succesed maka test koneksi sudah berhasil lalu pilih ok kemudian pilih ok lagi.
    Jika terjadi eror maka ada kesalahan pada penulisan direktorinya.
